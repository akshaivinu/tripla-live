Run the project locally in a few steps:

1. Install dependencies
npm install
2. Start the development server
npm run dev
3. Open in browser

Visit: http://localhost:3000


The platform is built as a modular, media-driven system, not a static page.

Core Stack
Next.js → Performance and routing
Framer Motion → UI transitions and interaction
GSAP → Advanced scroll and cinematic motion
Key Principles
Scene-based layout (not traditional sections)
Lazy-loaded media for performance
Scroll-triggered animations
Non-linear navigation
Experience Philosophy
